Скрипт считает количество элементов и приводит к удобному виду ПЭ3.

Формат входных данных - файл формата .csv, полученный путем копирования таблиц из перечня элементов ворда и
вствки их в эксель, с последующим сохранением в виде .csv.
Формат выходных данных - файл формата .csv. 

В переменную name_base_file вводить название файла, полученного из перечня.
В переменную name_write_file вводить название файла, куда хотим записывать результат.

В дериктории находятся файлы примера:
PE3.docx - исходный перечень элементов, оформленый по ГОСТ 2.702
test_input.csv - перечень, после приведения к нужному формату
test_output.csv - результат.
